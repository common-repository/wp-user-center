== Description ==

包括前台注册、登录、个人资料、每日签到、积分(包括积分商城、抽奖等)、查看隐藏内容（积分支付或者微信公众号验证码）、收藏、赞/踩、推广、我的文章等。

查看更多 https://wptao.com/wp-user-center.html

1. 前台注册、登录 （替换WordPress原有注册登录页面）
2. 前台资料页面 （将WordPress原有的个人资料页面搬到前台）
3. 积分计划和积分商城 （您的用户在进行一些操作时，奖励积分作为回报，可以兑换奖品/积分抽奖/积分支付-内容付费）
4. 查看隐藏内容（积分支付或者微信公众号验证码查看）
5. 每日签到领积分（如果需要微信签到，需要安装 WordPress连接微信 插件）
6. 收藏文章 （用户可以收藏您网站的文章，方便今后查阅。）
7. 文章评分 （赞或踩。用户可以对网站文章进行评分，以此表达一种态度，也可以匿名评分）
8. 推介链接 （已注册用户可以推荐新用户。在微信APP登录后分享给好友或者朋友圈自动加推介参数。）
9. 我的文章 （用户可以在前台查看自己的文章，其他用户也可以访问。）
10. 小工具（包括登录、签到、积分排行榜等）
11. 后台详细的积分记录和积分统计，数据分析一目了然。

== Installation ==

1. Upload the `wp-user-center` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the settings using "wp-user-center" under Settings panel.

下载插件，上传wp-user-center目录及其文件到 "/wp-content/plugins/" 插件目录，在后台管理中激活插件.
